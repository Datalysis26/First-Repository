## In dieser Rubrik geht es um meine praktischen Anwendungen von Machine-Learning-Methoden im Banken- und Versicherungsumfeld. Der Vorgang ist wie folgt gegliedert:
- Implementierung und Experimentieren mit verschiedenen ML-Algorithmen
- Fokus auf reale Anwendungsfälle und Datensätze
- Vergleich und Bewertung unterschiedlicher Modelle
- Datenvorverarbeitung, Feature Engineering und Modelloptimierung
- Visualisierung und Analyse der Ergebnisse
## 1. Bestimmung der Kreditausfallwahrscheinlichkeiten
- Hier erzeugte ich eine künstliche Datenbank und wendete hierauf sowie auf einer echten Datenbank aus OpenML die logistische Regression an, um Kreditausfallwahrscheinlichkeiten mithilfe der Sigmoid-Funktion zu bestimmen. Das Modell liefert unter der künstlichen Datenbank offensichtlich kein optimales Ergebnis, da die Daten zufällig generiert wurden. Unter der echten Datenbank aus OpenML liefert es jedoch ein befriedigendes Ergebnis. Die reale Datenbank stammt aus [https://www.openml.org/search?type=data&sort=runs&status=active&tags.tag=Data-Science&id=31].
